HTML

Hyper Text Markup Language is a markup language for web page creation.

Wow, he sure does look busy learning html.


Element Description
<html> Root element
<head> Contains metadata - title, scripts, style sheets, etc.
<body> Content of the html document
<h1> to <h6> 6 levels of section headings
<p> Paragraph
<header> Introductory content or navigation aids.
<footer> Typically has author info, copyright data or relevant links.
<main> Has main content for the body of the web page.
<nav> Section with navigation links
<section> Generic standalone section of website.
<search> Contains controls to perform a search or filter data.
